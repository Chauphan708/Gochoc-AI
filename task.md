# Danh sách nhiệm vụ dự án GócHọc AI (Phase 1)

Dựa trên kế hoạch phát triển MVP (Sản phẩm tối thiểu).

## 🟢 Cần thực hiện ngay
- [ ] Xác nhận phạm vi MVP (Sản phẩm tối thiểu) - Ưu tiên tính năng AI Bot và Dashboard cơ bản.
- [ ] Lựa chọn công nghệ chính thức (Đề xuất: Next.js + TailwindCSS cho PWA).
- [ ] Thiết kế UI Mockup cho Teacher Dashboard và Student Interface.
- [ ] Xây dựng Database Schema (PostgreSQL, Vector DB cho RAG).

## 📅 Lộ trình Phase 1 (4-6 tuần)

### Tuần 1-2: Nền tảng & Thiết kế
- [ ] Thiết kế UI/UX chi tiết.
- [ ] Thiết lập môi trường phát triển và Database schema.

### Tuần 2-3: Module Giáo viên
- [ ] Xây dựng tính năng tạo Phiên học (Session).
- [ ] Xây dựng tính năng thiết kế Góc/Trạm (nạp nội dung, nhiệm vụ).
- [ ] Module nạp tri thức (Knowledge Base) cho AI Bot.

### Tuần 3-4: AI Bot Service
- [ ] Triển khai RAG Engine (LangChain + Gemini/OpenAI).
- [ ] Lập trình logic cho Bot (đánh giá, hướng dẫn, trả lời trong phạm vi kiến thức).

### Tuần 4-5: Module Học sinh
- [ ] Xây dựng giao diện quét mã QR/nhập mã phiên.
- [ ] Giao diện tương tác với Bot tại trạm.
- [ ] Tính năng nộp bài (text/ảnh) và nhận phản hồi từ Bot.

### Tuần 5-6: Giám sát Real-time & Kiểm thử
- [ ] Kết nối Real-time Dashboard (GV giám sát HS).
- [ ] Kiểm thử toàn diện và thử nghiệm tại lớp học thực tế.
